# Number Guessing Game

A simple Python game where you try to guess a randomly generated number between 1 and 100.

## How to Play

1. Run the script:
    ```bash
    python game.py
    ```
2. Enter a number between 1 and 100.
3. The game will tell you if your guess is too high, too low, or correct.
4. Your score starts at 100 and decreases by 10 for each wrong guess.
5. Type `q` anytime to quit.
6. After winning, you can choose to play again or exit.

## Requirements

- Python 3.x
